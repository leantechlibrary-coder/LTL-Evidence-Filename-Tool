# LTL 証拠ファイル名変換ツール・7日試用版

証拠説明書の「号証」「標目」を貼り付けるだけで、フォルダ内のPDFファイル名を
mints提出形式（例：`甲001 標目.pdf`）にそろえる、訴訟実務向けの専用ツールの
7日間試用版です。

試用期間中は、有料版とまったく同じ全機能をお使いいただけます。

## 概要

- 試用期間：初回起動から7日間（フル機能）
- 試用期間終了後は、有料版をご購入ください
- 有料版：https://apps.microsoft.com/detail/9nr0xcfp1bj1?hl=ja-JP&gl=JP

## できること

- 証拠説明書の2列（号証・標目）を貼り付けるだけ。ファイル形式は問いません
  （スプレッドシート／Excel／Wordの表／CSVのいずれからコピーしても同じように動きます）。
- mints提出形式に正規化します（`甲001`、`甲002-1`、`乙A001` など。全角は自動で半角化）。
- 枝番束の結合：`甲002-1~2` のような範囲表記は、枝番PDFを1ファイルに結合します。
- 元ファイルは変更しません。結果は「（フォルダ名）_提出用」フォルダに新規出力されます。

## 使い方

1. 「① PDFフォルダを選択」で、号証PDFの入ったフォルダを開く
2. 証拠説明書の「号証」「標目」の2列を選んでコピーし、貼り付け欄に貼る
3. プレビューの色分けを確認する
4. 「リネーム実行」をクリック

## 動作環境

- Windows 10 / 11（64bit）
